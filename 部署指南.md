# 表盘部署指南（给小白的保姆级步骤）

目标：把表盘放到网上，你能改、别人能看。别人没有你的"管理员密码"就改不动。

预计耗时 **10~15 分钟**，全程在浏览器里点点点，不用装任何软件。

---

## 一次性准备

你需要一个 **Cloudflare** 账号（免费，不需要信用卡）。

1. 打开 https://dash.cloudflare.com/sign-up
2. 用邮箱注册，收邮件点激活链接。
3. 登录后，左边栏找 **Workers & Pages**（或 **Compute**）。

> 如果页面让你"添加域名"、"买域名"——全部跳过、忽略，我们不需要域名。

---

## 第 1 步：创建一个"数据保险柜"（KV Namespace）

这是用来存你那些链接数据的地方。

1. 左边栏 → **Storage & Databases**（或 **KV**）。
2. 点 **Create** → **KV Namespace**。
3. 名字填：`biaopan-data`
4. 点 **Add**。
5. 完成，先放着不管。

---

## 第 2 步：创建一个"小程序"（Worker）

这是真正对外服务的网页 + 后台。

1. 左边栏 → **Workers & Pages**。
2. 点 **Create** → **Workers** → **Create Worker**（或 **Hello World** 模板也行）。
3. 名字填：`biaopan`（或你喜欢的名字，只能用英文字母数字横线）。
4. 点 **Deploy**。页面会告诉你一个地址，类似：
   ```
   https://biaopan.你的用户名.workers.dev
   ```
   **这就是你以后打开表盘的网址。** 记下来。

---

## 第 3 步：把我写好的代码贴进去

1. 刚刚创建完 Worker 的页面，点 **Edit code**（或 **Quick edit**）。
2. 打开文件 `D:\AIcodeAPP\biaopan\worker.js`，用记事本/VSCode 都行，**全选**、**复制**。
3. 回到 Cloudflare 的代码编辑页，把原有内容**全选删除**，粘贴刚才复制的。
4. 点右上角 **Save and deploy** → **Save and deploy**（确认一下）。

---

## 第 4 步：把"保险柜"接到"小程序"上

Worker 必须被告知去哪个 KV 里存数据。

1. 退出代码编辑页，回到 Worker 的主页面（点左上角 Worker 名字 `biaopan`）。
2. 点 **Settings**（设置）→ **Bindings**（绑定）→ **Add**（添加）。
3. 选 **KV namespace**。
4. 两个字段：
   - **Variable name（变量名）**：`DATA`（**必须大写**，必须就是这几个字母）
   - **KV namespace**：从下拉选 `biaopan-data`
5. 点 **Deploy**（部署）。

---

## 第 5 步：设置管理员密码

这是你以后保存改动时要输入的密码。只有你知道。

1. 还在 Worker 的 **Settings** 页面 → **Variables and Secrets**（变量和密钥）→ **Add**。
2. **Type（类型）**：选 **Secret**（机密）。
3. **Variable name（变量名）**：`ADMIN_PASSWORD`（**必须完全一样**，大写加下划线）
4. **Value（值）**：你想设的密码，比如 `wo-de-biao-pan-2026`。自己记住。
5. 点 **Deploy**。

---

## 第 6 步：打开网址试一下

1. 浏览器访问第 2 步记下来的地址：
   ```
   https://biaopan.你的用户名.workers.dev
   ```
2. 应该看到一个黑色背景的空表盘，右下角写着"云端模式"。
3. 按键盘 **Q** 进入编辑模式。
4. 点 **+** 添加一个链接，比如 `github.com`。
5. 保存时会弹窗让你输入**管理员密码**，输第 5 步那个密码。
6. 输对就保存成功了，右下角显示"已同步"。
7. 刷新页面，链接还在，说明已经存到云上。

---

## 第 7 步：给别人分享 / 把桌面快捷方式指过去

- **分享**：把网址 `https://biaopan.你的用户名.workers.dev` 发给别人，他们打开就是你看到的样子。他们按 Q 也能点，但保存时没密码就存不进去。
- **桌面图标**：桌面上之前那个 `表盘.lnk` 打开的是本地文件。要让它打开云端的，告诉我你最终拿到的网址，我改一下指向。或者你自己：
  1. 右键桌面 → 新建 → 快捷方式。
  2. 位置填那个网址。
  3. 名字随意。

---

## 排错

| 问题 | 可能原因 | 处理 |
|---|---|---|
| 打开网址右下角红点"无法连接服务器" | KV 没绑定 | 回第 4 步，变量名必须叫 `DATA` |
| 保存时报"500" | KV 没绑定 或 密码没设 | 检查第 4 步和第 5 步 |
| 保存时一直让我输密码 | 密码变量名不对 | 第 5 步变量名必须叫 `ADMIN_PASSWORD`（全大写、带下划线） |
| 忘记密码 | 重设 | Settings → Variables and Secrets → 找到 `ADMIN_PASSWORD` → Edit → 改新值 → Deploy |
| 想把数据清空重来 | 删 KV 的 key | Storage → KV → biaopan-data → 删除 `items` |
| 在中国访问慢 | Cloudflare 在某些运营商不稳 | 等等再试；持续有问题告诉我，换个方案 |

## 以后怎么改表盘的外观或功能

- 如果只想改样式/功能（不改后端逻辑）：改 `D:\AIcodeAPP\biaopan\index.html`，告诉我，我帮你重新生成 `worker.js`，再把 worker.js 内容贴回 Cloudflare 的 Edit code 里重新 Deploy。
- **数据不会丢**——KV 里的数据和代码是分开的。
